# InLeaf – Plant E-Commerce Platform 🌿

**InLeaf** is a full-stack e-commerce web application developed as a final project for the **Web Programming** course at the University of Messina.

The platform allows users to browse plant-related products, manage a wishlist, add items to a shopping cart, and place orders. It also includes an **admin dashboard** for managing products, users, and orders.

The goal of the project was to build a clean, responsive, and functional online store while implementing core concepts of **web development, authentication, and database management**.

## Features

### User Features

- User registration and login with session authentication
- Product catalog with images, prices, and categories
- Wishlist system with toggle functionality
- Shopping cart with quantity control
- Checkout and order confirmation
- User profile with order history
- Product search and category filtering
- Responsive interface optimized for desktop and mobile

### Admin Features

- Admin authentication and protected dashboard
- Product management (Add / Edit / Delete)
- Image upload for products
- Order management
- User management
- Dashboard overview of store activity

## Project Structure

```
admin/      → Admin dashboard and product management
cart/       → Shopping cart and checkout logic
profile/    → User profile and order history
wish/       → Wishlist functionality
uploads/    → Uploaded product images

css/        → Custom styles
js/         → Client-side scripts

index.php   → Homepage
product.php → Product details page
login.php   → User login
register.php→ User registration
```
